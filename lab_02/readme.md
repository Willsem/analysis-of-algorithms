# Лабораторная работа 2

## Трудоемкость алгоритмов произведения матриц

### Build and Run

```
$ mkdir build
$ cd build
$ cmake -S .. -B .
$ make
$ ./lab_02 <size>
$ ./lab_02 <size_1> <size_2> <size_3> <size_4>
```
